## yelp!

#### A web application for search businesses and reviews.

### Features
* Search for businesses
* View a list of reviews
* Write a reviews
* Rate the businesses
* Add your businesses


### Technologies used:
* JavaScript
* HTML5
* CSS
* Bootstrap
